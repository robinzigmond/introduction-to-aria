ARIA fills the gaps

ARIA stands for Accessible Rich Internet Applications. It's a web standard designed to cover these gaps and enable developers to communicate their precise intent to assistive technologies (via the accessibility tree) when native HTML elements/attributes don't allow it

ARIA provides additional HTML attributes: the 
role
attribute and many 
aria-XXX
attributes, which are used to directly build the objects in the accessibility tree.

Like most web standards, it is continually evolving!

  • version 1.1 is supported in all modern browsers
  • version 1.2 reached "recommended" status in December 2021 but probably isn't sufficiently supported to use with confidence yet
  • version 1.3 is already in the planning stage!
